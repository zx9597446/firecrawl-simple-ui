# firecrawl-simple-ui

## to run this script

1. install uv：
    ```bash
    curl -LsSf https://astral.sh/uv/install.sh | sh
    ```
    windows user:
    ```bash
    powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
    ```

2. clone this repo：
   ```bash
   git clone https://github.com/zx9597446/firecrawl-simple-ui
   cd firecrawl-simple-ui
   ```

4. run with streamlit：
   ```bash
   uvx streamlit run src/app.py
   ```
